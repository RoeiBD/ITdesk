# ⚙️ ITDESK — מערכת ניהול קריאות IT

> ממשק וובי מקצועי לניהול קריאות תמיכה טכנית — Help Desk וסיסטם — עם עיצוב Dark Mode מלא.

## ✨ תכונות

- 🎫 **ניהול קריאות מלא** — פתיחה, מעקב ועדכון סטטוס עם מספר ייחודי לכל קריאה
- 👤 **פרטי מגיש** — שם, טלפון, אימייל ומחלקה לכל קריאה
- 🔍 **סינון וחיפוש** — לפי סטטוס, סוג ועדיפות בו-זמנית
- 📊 **סטטיסטיקות בזמן אמת** — סה״כ, פתוחות, בטיפול, קריטיות
- 🏷️ **סוגי קריאות** — HD (Help Desk) וסיסטם (System)
- 🔴 **עדיפויות** — קריטי / גבוהה / רגילה
- 🔔 **התראות Toast** בכל פעולה
- 📱 **ממשק נקי ומהיר** — פועל ישירות בדפדפן, ללא התקנה

---

## 🚀 התקנה והרצה

לא נדרשת התקנה. פשוט הורד את הקובץ ופתח בדפדפן:

```bash
git clone https://github.com/your-username/itdesk.git
cd itdesk
open index.html
```

או גרור את `index.html` לדפדפן ישירות.

---

## 🛠️ טכנולוגיות

| טכנולוגיה | שימוש |
|---|---|
| HTML5 | מבנה הדף |
| CSS3 | עיצוב, אנימציות, Dark Mode |
| Vanilla JavaScript | לוגיקת המערכת |
| Google Fonts (Rubik + JetBrains Mono) | טיפוגרפיה |

ללא תלויות חיצוניות — קובץ אחד בלבד.
